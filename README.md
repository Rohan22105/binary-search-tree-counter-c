# Binary Search Tree Counter
This is a C program that counts unique integer inputs and prints each value in sorted order with its frequency count using a binary search tree data structure.

## Features
- Counts duplicate integer values
- Stores data using a binary search tree
- Prints output in ascending sorted order
- Uses recursion and dynamic memory allocation
- Includes unit testing and Makefile support

## Skills Used
- C Programming
- Binary Search Trees
- Pointers
- Recursion
- Dynamic Memory Management
- Command-Line Arguments
- Makefiles

## Example

Input:
```bash
./unique 314 95064 76 76 95064 76
```

Output:
```bash
3 76
1 314
2 95064
```

## Files
- `unique.c` — Main program logic
- `tree.c` — Binary search tree implementation
- `tree.h` — Tree structure definitions
- `unittests.c` — Unit tests
- `Makefile` — Build configuration


## Run

```bash
./unique
```
